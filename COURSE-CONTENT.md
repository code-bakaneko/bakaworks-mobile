# Course Content

Live contents of the `lesson_sets` tree, with room for your notes.

## How to leave notes

Write any line starting with `>` directly **underneath** an anchor. Anchors look
like `[item:42]`, `[set:3.2]` or `[lesson:3]` and never change.

```
- [item:42] listen — plays か → か / き / く / け
  > swap this for a harder distractor
  > and move it after the trace
```

Then run `node scripts/dump-content.js` — the file rebuilds from the database
and **your notes come back with it**, attached to the same anchors.

Notes on something that does not exist yet go in the Requests section at the
bottom, which is preserved as a whole.

##### A Row &nbsp; `[lesson:1]`
_a i u e o_

**Set 1** &nbsp; `[set:1.1]`

- `[item:8]` **lecture** — Japanese is written with three scripts at once. Hiragana is the rounded, flowing one, and it can spell every sound in the language. Katakana handles foreign words. Kanji are borrowed Chinese characters that carry meaning. You start with hiragana, because everything else is built on top of it.
- `[item:9]` **lecture** — The first five hiragana are the vowels: あ (a), い (i), う (u), え (e), お (o). Every other hiragana character is one of these vowels with a consonant attached. Learn these five and you already know the ending of all 46.

**Set 2** &nbsp; `[set:1.2]`

- `[item:65]` **trace** — あ (a), 3 strokes

**Set 3** &nbsp; `[set:1.3]`

- `[item:77]` **listen** — plays `あ` → **あ** / い / う / え
- `[item:78]` **listen** — plays `い` → お / **い** / う / あ
- `[item:79]` **listen** — plays `う` → え / あ / **う** / お
- `[item:80]` **listen** — plays `え` → **え** / い / お / あ
- `[item:81]` **listen** — plays `お` → う / あ / **お** / い

##### Wa Row &nbsp; `[lesson:10]`
_wa wo n_

**Set 1** &nbsp; `[set:10.1]`

- `[item:192]` **lecture** — The last three: わ, を, ん. を is special — it is only ever used as a grammar particle marking the object of a sentence, never inside a word, and it is usually pronounced just 'o'. ん is the only character that is a consonant on its own, and the only one that can never start a word.
- `[item:193]` **listen** — plays `わ` → **わ** / を / ん / ね
